# Security Policy

## Reporting a Vulnerability

- Не создавайте публичные issues по уязвимостям.
- Напишите на security@yourdomain.com с подробным описанием:
  - версии ПО и окружения
  - шагов воспроизведения
  - ожидаемого/фактического поведения
  - PoC (если есть)
- Мы подтвердим получение и вернёмся с оценкой и планом фикса.

## Supported Versions

- Поддерживаем только ветку `main` текущего репозитория.

## Disclosure

- Ответственно координируем раскрытие и укажем вас в благодарностях, если пожелаете.
